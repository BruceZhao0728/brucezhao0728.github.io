Other Projects

SJTU

Vulnerability Analysis System Based on Vector Databases and Artificial Intelligence, Sep.2025 - Oct.2025

I was responsible for dealing with generating SQL queries from natural language, automatically analyzing and constructing slices of input programs using Tree-Sitter, embedding slices into high-dimensional vectors with GraphCodeBERT and CodeT5+, and performing vulnerability matching to identify potential CVEs and CWEs in a vectorized vulnerability database.


Kernel-Module-Based Packet Filtering Firewall, Jun.2025 - Jul.2025

As the team leader, I designed and implemented the kernel module using Netfilter via C++ and built the rule management system backed by SQLite.


Design of a New Distributed Power Coordinated Controller, Oct.2023 - Oct.2024

I was responsible for the coordination of multiple generators and loads by the consensus algorithm and the optimization of the communication topology by the genetic algorithm via Python.
