Research Projects | 研究项目

AI Software Protection System Based on Code Obfuscation

基于代码混淆的人工智能软件保护系统

04/2024 - 05/2026

Project Description | 项目简介

  • Designed an automated code protection based on TVM and LLVM IR to shield localized AI models from reverse engineering, preventing unauthorized extraction of proprietary model architecture.
    针对 AI 模型本地部署后，其二进制文件易被逆向工程提取模型结构与权重，导致知识产权泄露的行业痛点，开发基于混淆的代码保护系统，在不影响模型精度和性能的前提下，将逆向工程成本提升至经济不可行，使本地人工智能软件免受逆向分析等手段的攻击，保护模型开发者的知识产权。
  • Pioneered the theoretical expansion of Mixed Boolean-Arithmetic (MBA) from the integer domain to the floating-point domain, successfully obfuscating core AI operations (e.g., Conv) while maintaining 100% numerical consistency.
    将传统混合布林代数（MBA）从整数域突破性拓展至浮点数域，攻克了传统方法无法保护AI模型核心浮点计算的难题，提出并证明了混淆变换的正确性，确保混淆前后模型输出100%一致。
  • Engineered a custom LLVM Pass to implement multi-level obfuscation during TVM compilation, allowing for tunable parameters such as recursion depth and MBA complexity to balance performance overhead and security strength.
    基于 TVM 机器学习编译器和 LLVM 编译器架构，在 LLVM IR 中间表示层设计并实现了自动化混淆 Pass ，支持用户自定义混淆参数，生成可直接部署的混淆版二进制库/可执行文件，实现了性能开销与安全强度的灵活平衡。
  • Quantified efficacy by developing a standardized metric system; experimental results showed a >30% decrease in de-obfuscation success rates and a >40% decrease in reverse engineering success rates compared to SOTA works.
    浮点数表达式和 AI 模型运算结果在混淆前后100%一致，反混淆工具成功率下降30%以上，逆向工程成功率下降40%以上，同时提出统一的混淆效果量化指标体系，衡量代码混淆的强度和安全性。
  • Authored a research paper detailing the implementation.
    作为共同第一作者和项目负责人完成论文。

Tech Stack | 技术栈
C++, LLVM, TVM, Reverse Engineering, Code Obfuscation

AI-Powered Vulnerability Analysis System Based on Vector Databases

基于向量数据库和人工智能的智能漏洞分析系统

09/2025 - 10/2025

Project Description | 项目简介

  • Architected a hybrid vulnerability detection system utilizing Chroma (Vector DB) and MySQL (Relational DB).
    采用双模数据库架构，结合关系型数据库 MySQL 和向量数据库 Chroma 的优势，实现结构化数据管理和语义相似性检索的完美结合。
  • Implemented program slicing and static analysis to parse ASTs and extract data-flow/control-flow dependencies.
    结合程序切片技术，使用 Tree-Sitter 解析不同程序语言，对抽象语法树（AST）进行程序切片，提取程序中的数据流和控制流依赖关系。
  • Developed a semantic-structural embedding engine utilizing CodeT5+ and GraphCodeBERT to export vectors of program slices.
    利用 CodeT5+ 模型捕捉程序深层语义，利用 GraphCodeBERT 模型捕捉程序结构，生成含语义与结构双重特征的切片向量，识别潜在漏洞。
  • Engineered a natural language interface that automates SQL query generation via cloud-based LLM APIs.
    构建自然语言接口，用户输入查询需求，系统通过云端 LLM API 自动生成 SQL 查询语句，实现对关系型数据库的智能查询。

Tech Stack | 技术栈
Python, SQL, Chroma, Tree-Sitter, Program Analysis

Design of a New Distributed Power Coordinated Controller

新型电力系统分布式协调控制器设计

10/2023 - 10/2024

Project Description | 项目简介

  • Architected a decentralized "Cloud-Edge" collaborative control framework to address modeling complexities in modern power systems.
    设计一套去中心化的“云-边”协同控制架构，应对新型电力系统中海量分布式资源与电/气/热多能流耦合带来的建模与控制复杂性挑战。
  • Developed a distributed consensus algorithm based on Multi-Agent Systems and Graph Theory, featuring global convergence and power balancing.
    基于图论与多智能体系统理论设计了一致性变量的更新律，保证一致性算法收敛性和功率平衡特性，提升系统稳定性与鲁棒性。

    基于遗传算法，以通信拓扑网络的连接矩阵为染色体，筛选出抗延时能力最强、可靠性最强的上层网络拓扑结构，提高网络性能。
  • Implemented the distributed architecture on Raspberry Pi hardware using TCP/IP protocols for real-time data exchange.
    使用树莓派开发基于 TCP 协议的通信网络，实现上层云平台、下层微电网控制器的分布式架构。
